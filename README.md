# Theme Cluster

Custom WordPress theme built with:

- WordPress
- ACF
- SCSS
- JavaScript
- Gulp

## Development

Frontend assets are generated from the source project and compiled into the `assets` directory.

### Features

- Semantic HTML
- BEM methodology
- SCSS
- JavaScript modules
- Gulp
- Responsive layout
- Cross-browser support

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Frontend Structure

- `src/pages` — page templates
- `src/layouts` — layouts and partials
- `src/blocks` — page sections
- `src/elements` — UI elements
- `src/scss` — styles
- `src/js` — JavaScript modules
- `dist` — compiled files

## Theme Structure

- `inc/` — theme setup and configuration
- `assets/` — compiled CSS, JS, images and fonts
- `front-page.php` — homepage template

## Requirements

- WordPress 6+
- PHP 8.1+
